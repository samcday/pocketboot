# Pocketboot attic

This directory preserves experiments that produced useful engineering evidence
but are not part of Pocketboot's supported architecture. An attic report is a
handover, not a feature promise: follow its pinned branch or tag to resume the
work, and re-evaluate its assumptions before using it on hardware.

## Archived experiments

- [Crosshatch stockOS chainload, July 2026](crosshatch-stockos-chainload-2026-07/README.md)

Raw device captures should not be committed here. Keep only small, redacted
excerpts plus hashes and provenance for any separately retained originals.
